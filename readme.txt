Add a cookie consent bar in your website.

== Description ==

The KJ Rocker Cookie Consent WordPress plugin is a straightforward solution to make websites compliant with GDPR and EU cookie legislation. It notifies visitors about the cookie policy, directs them to the policy page for any concerns, and obtains their consent.

== Features ==
- Seamless Integration.
- Optimized for Conversions: Non-intrusive, ensuring user engagement.
- GDPR & EU Compliant.
